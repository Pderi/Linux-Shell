## 3. P1 — 内建命令实现

> 顺序：cat（最简单）→ pwd → ls → grep → 调度确认

### P1-01 实现 builtin_cat

| 项 | 内容 |
|----|------|
| **文件** | `src/builtin_cat.c` |
| **功能** | |
| | - 无参数：stdin → stdout 循环 copy |
| | - 有参数：`open(O_RDONLY)` → read/write 循环 → close |
| | - 多文件依次处理，失败 perror 并 rc=1 |
| **系统调用** | `open`, `read`, `write`, `close` |
| **验收** | 逻辑审查通过；联调后 `cat /etc/passwd` 含 root |
| **依赖** | P0-03 |

### P1-02 实现 builtin_pwd

| 项 | 内容 |
|----|------|
| **文件** | `src/builtins.c`（函数体放此或独立，统一注册即可） |
| **功能** | `getcwd(buf, sizeof(buf))` → puts |
| **系统调用** | `getcwd` |
| **验收** | `cd /tmp && pwd` 输出 `/tmp` |
| **依赖** | P0-03 |

### P1-03 实现 builtin_ls

| 项 | 内容 |
|----|------|
| **文件** | `src/builtin_ls.c` |
| **功能** | |
| | - `ls` / `ls path`：`opendir` → `readdir` → 输出 |
| | - `-l`：`stat` 输出权限/链接数/大小/时间/名 |
| | - 单文件路径（如 `ls /bin/ls`）：stat 非目录则直接输出路径 |
| | - 跳过 `.` 开头（除非后续加 `-a`） |
| **系统调用** | `opendir`, `readdir`, `closedir`, `stat` |
| **验收** | `ls /tmp` 有输出；`ls /bin/ls` 含 `/bin/ls` |
| **依赖** | P0-03 |

### P1-04 实现 builtin_grep

| 项 | 内容 |
|----|------|
| **文件** | `src/builtin_grep.c` |
| **功能** | |
| | - `grep pattern file`：逐行读，子串匹配输出 |

## 4. P2 — 执行器改造

> 核心：删 execvp，修 waitpid，管道只允许内建

### P2-01 删除 execvp，非内建报 not found

| 项 | 内容 |
|----|------|
| **文件** | `src/executor.c` |
| **改动** | |
| | 1. `run_command_child`：删 `execvp`，非内建 → `command not found` + `_exit(127)` |
| | 2. `execute_single` 入口：非内建直接报错返回 127 |
| | 3. `execute_pipeline` 入口：遍历每段，非内建报错返回 127 |
| | 4. 全项目 grep 确认无 `execvp`/`system`/`popen` |
| **验收** | 源码无 exec；`wc -l` 类命令报 not found |
| **依赖** | P1-05 |

### P2-02 修复 waitpid 竞态（内建 + 重定向）

| 项 | 内容 |
|----|------|
| **文件** | `src/executor.c` |
| **改动** | |
| | 1. 新增 `run_builtin_in_parent(cmd)`：save fd → dup2 → run_builtin → restore fd |
| | 2. `executor_run_single_builtin`：前台 + 有重定向 → 调 `run_builtin_in_parent`，**不 fork** |
| | 3. `wait_pid_blocking`：`errno == ECHILD` 时视为已回收（兜底） |
| **验收** | `echo test > /tmp/out.txt` 无 `waitpid: No child processes` |
| **依赖** | P2-01 |

### P2-03 管道 + 内建联调

| 项 | 内容 |
|----|------|
| **文件** | `src/executor.c`（必要时微调） |
| **验证命令** | |
| | - `echo hello \| grep hello` |
| | - `grep root /etc/passwd \| grep root` |
| | - `echo first > f && echo second >> f && cat < f` |
| | - `grep root /etc/passwd &` |
| **验收** | 以上命令输出/行为正确，无 waitpid 报错 |
| **依赖** | P2-02, P1-01~04 |

## 10. 失败时的定位指南

| 失败现象 | 优先查 | 对应任务 |
|----------|--------|----------|
| `command not found` 误报 | `type.c` 内建表、`is_builtin` | P0-02, P1-05 |
| `waitpid: No child processes` | `executor_run_single_builtin` 是否仍 fork | P2-02 |
| 管道无输出 | `dup2` 顺序、子进程 fd 关闭 | P2-03 |
| `grep` 管道不工作 | stdin 是否从 fd 0 读 | P1-04 |
| `ls -l` 格式不对 | `builtin_ls.c` stat 输出 | P1-03 |
| `alias dir='ls -l'` 失败 | `alias.c` argv 合并 | P3-03 |
| Tab 不补全 | `completion.c` | P3-02 |
| 测试 T5 失败 | 脚本是否仍用 wc | P4-01 |
| 编译失败 | Makefile SRCS、头文件声明 | P0-02 |
